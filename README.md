# Project Reboot – 180 Days to Become an AI Software Engineer

## About

This repository documents my 180-day journey of rebuilding my Computer Science skills after a long career break from software development.

I graduated with a B.Tech in Computer Science in 2011 and worked for 12 years as a Pharmacy Store In-charge. In 2026, I decided to restart my software engineering career with the goal of becoming an AI Software Engineer.

This repository contains my daily practice, projects, notes, and progress.

## Goals

- Master Python

- Learn Git & GitHub
- Learn AI and Machine Learning
- Build real-world projects
- Become job-ready within 6 months
