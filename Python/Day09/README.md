## 💻 Project

### Student Management System

A modular command-line application that manages student records and generates a class report.

### Features

- Accepts multiple student names
- Accepts marks for each student
- Validates all user inputs
- Calculates:
  - Total Marks
  - Average Marks
  - Highest Marks
  - Lowest Marks
- Supports multiple toppers
- Displays a formatted class report
