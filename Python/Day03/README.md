## 💻 Mini Projects Completed

### ✅ 1. Age Eligibility Checker
Checks whether a user is eligible to vote based on age.

### ✅ 2. Pass/Fail Checker
Accepts student marks and assigns grades:
- Grade A
- Grade B
- Grade C
- Fail

Includes input validation for invalid marks.

### ✅ 3. Even/Odd Checker
Determines whether a number is even or odd using the modulo (`%`) operator.

### ✅ 4. Positive/Negative/Zero Checker
Identifies whether a number is:
- Positive
- Negative
- Zero

Includes validation for incorrect inputs.

### ✅ 5. Simple Login System
A basic username and password verification program using conditional statements.

### ✅ 6. Password Strength Checker v2
Checks password strength based on password length.

Levels:
- Weak
- Medium
- Strong

Includes validation for blank passwords and spaces.

### ✅ 7. Number Guessing Game (Version 1)
The user guesses a secret number.

The program responds with:
- Too Low
- Too High
- Correct Guess
